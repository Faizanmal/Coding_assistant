## 📥 Installation

### From Marketplace (Coming Soon)

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Copilot - AI-Powered Coding Assistant"
4. Click Install

### Manual Installation (Development)

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Faizanmal/Coding_assistant.git
   cd Coding_assistant
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Build the Extension**
   ```bash
   npm run compile
   ```

4. **Set Up Environment Variables**
   
   Create a `.env` file in the root directory:
   ```bash
   cp .env.example .env
   ```
   
   Edit `.env` and add your API keys:
   ```env
   # AI Provider API Keys (add at least one)
   GROQ_API_KEY=your_groq_api_key_here
   TOGETHER_API_KEY=your_together_api_key_here
   OPENROUTER_API_KEY=your_openrouter_api_key_here
   MISTRAL_API_KEY=your_mistral_api_key_here
   CEREBRAS_API_KEY=your_cerebras_api_key_here
   
   # Optional
   TAVILY_API_KEY=your_tavily_key_here
   HUGGINGFACE_API_KEY=your_huggingface_token_here
   ```

5. **Start Backend Server** (Optional but recommended)
   ```bash
   cd backend-server
   npm install
   npm start
   ```

6. **Run in Development Mode**
   - Press F5 in VS Code to launch extension development host
   - Or run: `npm run watch` for auto-rebuild

7. **Package Extension** (Optional)
   ```bash
   npm run package
   ```
   This creates a `.vsix` file you can install manually.

## 🚀 Quick Start

### 1. First Launch

After installation:

1. **Open Command Palette**: `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS)
2. **Type**: `Coding` to see all available commands
3. **Open AI Chat**: Select `Open AI Chat` or use sidebar icon

### 2. Configure AI Provider

1. Click on the **AI Chat** sidebar icon
2. Select your preferred **AI Provider** from dropdown
3. Choose an appropriate **Model** for your task
4. Start chatting with AI!

### 3. Try Your First Command

**Generate Code from Description:**
```
1. Press Ctrl+Shift+P
2. Type: "Generate Code from Description"
3. Describe what you want: "Create a React login component with validation"
4. Watch AI generate production-ready code!
```

**Run Security Scan:**
```
1. Press Ctrl+Shift+P
2. Type: "Enhanced Security Scan - Workspace"
3. Review security findings in dashboard
4. Apply automated fixes
```

**Start Learning Journey:**
```
1. Press Ctrl+Shift+P
2. Type: "Start Learning Journey"
3. Choose your learning goal
4. Follow personalized learning path
```

### 4. Quick Shortcuts

| Shortcut | Action |
|----------|--------|
